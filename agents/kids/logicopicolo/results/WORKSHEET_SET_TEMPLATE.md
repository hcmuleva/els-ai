# Worksheet Set Template

## Required Files
- `<worksheet_id>_front.html`
- `<worksheet_id>_flip.html`

## Minimum Metadata in Each File
- worksheet_id
- topic
- structure_version: LP_FIXED_V1
- generation_date

## Flip Side Requirement
Flip side must mirror the front layout:
- Same randomized left card grid as front side, with fixed slot buttons
- Same right option display order as front side
- Each right option row must include the corresponding correct slot button

## Front Side Requirement
Front side left card grid must be randomized in visual order (not S1..S10 sequence).
Slot ID and fixed slot button per card must remain unchanged.

Use the same fixed 10 buttons on both sides so frame alignment stays consistent.
